MQTT-Based IoT Control App

This project is an MQTT-based IoT control application designed to manage various devices and actions such as up, down, stop, program, cartun, lamp, door, light on, and light off. The app provides a user-friendly interface to control IoT devices seamlessly over an MQTT protocol.

Main Features:

  • Device Control: Control devices like lights, doors, and motors with actions such as on, off, up, down, and stop.
  • Programmable Actions: Set up programmable actions (prog) for automated tasks.
  • Real-Time Monitoring: Monitor device status in real-time.
  • User-Friendly Interface: Intuitive design for easy navigation and control.
  • MQTT Protocol: Efficient and lightweight communication protocol for IoT devices.

This app is ideal for smart home automation, industrial IoT, and other applications requiring remote device control and monitoring.
